# Estudando Express-JS com a Mozilla org.



Projeto de desenvolvimento front-end para a construção de um webapp de busca de restaurantes utilizando a biblioteca React-JS e a APIs de busca do Google. [^1]



Dependências:

- Express-JS
- Morgan
- MongoDB
